# Engineering Arena — STEM Games 2026

## The task

Rather than a single design challenge, the Engineering arena was a series of
**20+ short, interactive tasks** spanning many engineering disciplines —
control theory, signal processing, structural mechanics, thermodynamics, and
more. Each task is solved through an interactive web page, and each completed
task also required uploading a document showing the work and calculations.

Topics seen across the tasks include: LQR control, cross-correlation / radar
convolution, Kalman filtering, Mohr's circle, beam deflection, PID tuning,
forward kinematics, FFT spectral analysis, Euler buckling, Rankine cycles,
Nyquist stability, and torsional stress.
