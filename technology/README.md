# Technology Arena — STEM Games 2026

**Result: 6th place.**

The Technology arena ran over two days:

- **Day 1 — competitive programming.** A set of algorithmic problems solved
  and submitted on an online judge. See [`day1-leetcode/`](day1-leetcode/).
- **Day 2 — project.** A single open-ended photogrammetry problem: estimating
  3D point clouds from photographs. See [`day2-project/`](day2-project/).

The Day 2 project scored **400 / 600**. The Day 1 algorithmic round scored
**10 / 400** (later scaled by 3/4), contributing a smaller share of the total.
