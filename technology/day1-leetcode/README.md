# Technology — Day 1: Competitive Programming

The Day 1 round consisted of algorithmic problems judged online. The problems:

- **Undistortion** — invert a radial lens-distortion formula.
- **Largest Triangle** — find the maximum-area triangle over a set of points.
- **Gaussian Blur** — weighted sums over square neighbourhoods, modulo a prime.
- **Find the Square** — locate a white square in a grayscale image.

## Contents

| Folder / file | What's inside |
|---------------|---------------|
| `problem-statements/` | Official problem statements (PDF). |
| `solutions-competition/` | Solutions written by the team **during the competition** (currently the partial Gaussian Blur solution scored 10/100). |
| `solutions-ai-postcomp/` | Solutions produced **after the competition** with AI assistance, added as reference only. **Not** part of the competition submission. |
| `testcases/` | The grading test clusters (input/output pairs) for each problem. |
| `scripts/` | Helper script(s) for downloading the test data so it does not have to be collected by hand. |
| `Results.pdf` | The Day 1 round results / scoreboard. |

> **Integrity note:** keep competition work and post-competition AI-assisted
> work in their separate folders. Anyone studying this archive should be able
> to tell at a glance which is which.
